# Evaluacion Unidad 3

## Proyecto

- SPA React: gestor de proyectos estudiantiles.
- CRUD: agregar, mostrar, editar y eliminar.
- Persistencia: `localStorage`.
- Estilos: Bootstrap y Bootstrap Icons.

## Componentes

- `src/App.jsx`
- `src/components/ProjectForm.jsx`
- `src/components/ProjectTable.jsx`
- `src/components/ProjectRow.jsx`

## Instalacion

```bash
pnpm install
```

## Ejecucion

```bash
pnpm run dev
```

## Compilacion

```bash
pnpm run build
```

## Scripts

- `pnpm run dev`: servidor local.
- `pnpm run build`: version de produccion.
- `pnpm run preview`: previsualizacion de produccion.
